# StockChef Platform

Plataforma full stack para gestão de estoque de restaurantes, desenvolvida com foco em arquitetura escalável, boas práticas de engenharia de software, observabilidade, mensageria e experiência corporativa.

## Objetivo do projeto

O StockChef Platform foi idealizado como um sistema de nível profissional para controle de estoque de restaurantes e redes de alimentação, permitindo gerenciar produtos, movimentações, compras, inventários, perdas, fichas técnicas e alertas operacionais.

O objetivo é construir uma aplicação que demonstre domínio técnico em desenvolvimento full stack com .NET e Angular, seguindo uma abordagem próxima de ambientes empresariais reais.

## Principais funcionalidades planejadas

- Cadastro de produtos e insumos
- Controle de estoque por unidade
- Movimentações de entrada e saída
- Gestão de fornecedores
- Compras e recebimento
- Inventário e ajuste de estoque
- Controle de perdas e desperdícios
- Fichas técnicas de pratos
- Baixa automática de insumos por receita
- Alertas de estoque mínimo e vencimento
- Dashboard gerencial
- Auditoria de operações

## Arquitetura planejada
